# copaw-worker

Lightweight [AgentTeams](https://github.com/higress-group/hiclaw) Worker runtime based on [CoPaw](https://github.com/agentscope-ai/CoPaw).

## Install

```bash
pip install copaw-worker
```

## Usage

```bash
copaw-worker --name <worker-name> --fs <minio-endpoint> --fs-key <access-key> --fs-secret <secret-key>
```

See [AgentTeams worker guide](https://github.com/higress-group/hiclaw/blob/main/docs/worker-guide.md) for full setup instructions.
